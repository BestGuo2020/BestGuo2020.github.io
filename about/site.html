关于我的小站

2021-01-01- 2022-05-25

1、目的和意义

分享自己的学习成果，经验总结以及日常。

2、历史

小赫赫的小站，前身叫 BestGuo。大专时期，在 2018 年年底，利用 Django 写了一个自己的个人网站。

当时正读大二，而且那个学期在学习 Django 和 Bootstrap，就拿来练练手。所以当时的页面就非常之简单

这是我的旧网站，哈哈几年前自己写的就是很丑

然后有一个打算，就是在暑假的自己重写自己的个人网站，当时是在用 vue 进行重写，使用 element-ui 的 vue 前端框架去编写。写完之后的结果发现由于组件之间的耦合性太强了，因为组件之间基本是子传父，父传子，而且当时也不会用 Vuex，所以这个前端页面就直接成为了废案。

这是之前的废案

然后前端变为废案，就从后端开始做，后端弄完再用 vue 写一个前端，后端弄完，由于在本科阶段时间很紧，根本没时间，再加上一个 Java Web 的期末项目制作。就这样被搁置了。

2021年1月1日之前，想约一个女孩子出去玩😂，但却被其她女生约走了😭😭，现在元旦一个人去玩又很无聊，索性就把个人网站搭建起来。后面我就想起了使用 Hexo 来搭建，在使用 Hexo 搭建博客发现，它真的是太强了。

Hexo，永远滴神！👍👍

现在，网站名也改为“小赫赫的小站”😄😄

2021-11-14

想了想，网页名称改，叫 BestGuo 的小窝吧😄😄。
